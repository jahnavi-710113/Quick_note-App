# Quick_note-App
What is Quick Notes? Quick Notes is a full-stack web application that lets users write, save, and manage personal notes — all on a single page. Notes are stored permanently in a database and instantly appear on the same page without refreshing. 
notes-app/
├── backend/
│   ├── main.py        ← FastAPI server + all API routes
│   └── notes.db       ← SQLite database (auto-created)
└── frontend/
    └── index.html     ← Complete UI (HTML + CSS + JS)
